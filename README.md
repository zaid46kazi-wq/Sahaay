## 🚀 Overview

Sahaay is an AI-powered caregiver support system that detects emotional stress and provides real-time, non-intrusive support through conversation.

---

## 🎯 Problem

Caregivers experience hidden stress, burnout, and emotional fatigue but lack time and safe spaces to seek help.

---

## 💡 Solution

Sahaay acts as a silent AI companion that:

* Detects emotions through chat/voice
* Provides empathetic responses
* Suggests small stress-relief actions
* Tracks stress patterns over time

---

## 🌐 Sahaay Community

Sahaay is not just AI—it is a **support ecosystem**.

* Anonymous safe space for caregivers
* Share thoughts, struggles, and experiences
* Peer-to-peer support
* Reduces loneliness and emotional isolation

👉 *Because sometimes, the best support comes from someone who understands.*

## 🚀 Deployment Guide

### 1. Clone Repo

```bash id="6x4n8g"
git clone https://github.com/YOUR_USERNAME/sahaay-ai-caregiver-support.git
cd sahaay-ai-caregiver-support
```

---

### 2. Install Dependencies

```bash id="w1slbh"
npm install
```

---

### 3. Setup Environment Variables

Create `.env` file:

```env id="yoq6cx"
GEMINI_API_KEY=your_key_here
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_KEY=your_key
```

---

### 4. Run Locally

```bash id="t82o2u"
npm run dev
```

---

### 5. Deploy

* Frontend → Vercel
* Backend → Render / Railway
* Database → Supabase
